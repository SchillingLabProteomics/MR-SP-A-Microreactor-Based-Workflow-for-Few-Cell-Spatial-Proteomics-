# MR-SP-A-Microreactor-Based-Workflow-for-Few-Cell-Spatial-Proteomics-
Analysis code and processed data for evaluating MR-SP², a microreactor-based workflow for spatial proteomics of LMD-isolated FFPE tissue. Includes notebooks, environment files, and result tables supporting comparison of MR-SP² vs tube-based prep across few-cell microregions.
